# Fieldset
`mwc-fieldset`

## Examples
### Normal
```html
<mwc-fieldset>
    <span slot="legend">Header</span>
    Hello content
</mwc-fieldset>
```
<mwc-fieldset>
    <span slot="legend">Header</span>
    Hello content
</mwc-fieldset>

### Bordered
```html
<mwc-fieldset bordered>
    <span slot="legend">Header</span>
    Hello content
</mwc-fieldset>
```
<mwc-fieldset bordered>
    <span slot="legend">Header</span>
    Hello content
</mwc-fieldset>

## Slots
| Name | Description |
| --- | ---|
| [default] | |
| legend | Heading of the fieldset |

## Attributes
| Name | Value | Description |
| --- | --- | --- |
| bordered | | Applies more visible border |
